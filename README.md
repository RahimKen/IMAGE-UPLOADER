
# Project Title

Image Uploader App using React js - Tailwindcss - Node js - Express js


## How to Run it
Firstly you need to open two Terminals in order to run the frontend and the backend in the same time .

#### run the frontend 

```bash
  cd frontend
  npm install
  npm run start
```

#### run the backend

```bash
  cd backend
  npm install
  npm run start

```
